# Empower Mind

A static Astro foundation for a distinctive NGO and youth mental-health website, prepared for Codex, Kiro, and Cloudflare Pages.

## Local development

```bash
pnpm install
pnpm dev
```

## Validate

```bash
pnpm check
pnpm lint
pnpm test
pnpm build
```

## Deploy to Cloudflare Pages

```bash
pnpm deploy
```

Project name: `empower-mind`  
Output directory: `dist`  
Provisional URL: `https://empower-mind.pages.dev`

## Agent context

- Codex begins with `AGENTS.md`.
- Kiro begins with `.kiro/prompts/start-empower-mind.md`.
- Organization truth lives in `project-content/`.
- Missing content must remain in `project-content/content-gaps.md` rather than being invented.
