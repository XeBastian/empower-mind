# Logo assets

`logo.jpg` is the organization logo supplied for the website and remains the source reference.

`logo-transparent.svg` preserves the supplied artwork inside an SVG and removes the white field with an alpha filter. The header uses this transparent version so the mark blends with the site surface.

The transparent SVG is a fidelity-preserving web derivative, not a manually redrawn vector master. Replace it if original path-based brand artwork becomes available. Keep source files and optimized web derivatives distinct, and update the asset register with ownership and approved-use information.
